# 把 AI 装进口袋：OpenClaw 深度体验指南

> 想象一下：你在地铁上，突然想起需要查个资料。不用打开电脑，不用登录网页，直接在 WhatsApp 里给 AI 发条消息，几秒钟后答案就回来了。

这就是 OpenClaw 能做的事。

---

## 一、OpenClaw 到底是什么？

### 用大白话说

OpenClaw 就是一个**桥梁**，把你天天用的聊天软件（WhatsApp、Telegram、Discord）和 AI 智能体连接起来。

**没有 OpenClaw 时：**
```
你想用 AI → 打开浏览器 → 找到 AI 网页 → 登录 → 输入问题 → 等待回复
```

**有了 OpenClaw 后：**
```
你想用 AI → 打开微信/WhatsApp → 发消息给 AI → 等待回复
```

少了好几步，对吧？

### 为什么叫"Claw"（爪子）？

官方有个有趣的说法：**OpenClaw = CLAW + TARDIS**（时空机器）。意思是这只"太空龙虾"能帮你跨越时间和空间获取信息 🦞

好吧，这个梗可能只有开发者懂，但不影响它好用。

## 二、它能帮你做什么？

### 场景 1：随时随地的私人助手

```
你：今天上海天气怎么样？
AI：上海今天晴，15-22°C，适合出门，记得带件薄外套。

你：帮我查一下特斯拉最新股价
AI：特斯拉当前股价 $248.50，今日上涨 2.3%...
```

**关键点**：你不需要打开任何 AI 网页，就像给朋友发消息一样自然。

在公司 Discord/Slack 群里：

```
同事 A：@AI 帮我把上周的会议纪要整理一下
AI：好的，正在查找... 已找到 3 份会议纪要，已整理成文档：[链接]

AI：已创建任务"项目交付"，截止日期 2026-03-06，负责人待分配。
```

**不用开电脑，不用 SSH 连接**，几条消息搞定。

## 三、它是怎么工作的？

### 核心架构（简化版）

```
你的手机 (WhatsApp/Telegram)
        ↓
    OpenClaw 网关（运行在你家/公司的电脑上）
        ↓
    AI 智能体（处理你的请求）
        ↓
    各种工具（搜索、文件、代码执行等）
```

**关键点**：
- OpenClaw 运行在**你自己的设备**上（不是云端服务）
- 数据不经过第三方，隐私更有保障
- 你可以完全控制它能做什么、不能做什么

### 为什么需要"网关"？

想象网关就是一个**24 小时待命的管家**：
- 它时刻监听你的消息
- 收到消息后判断要不要处理
- 需要时叫醒 AI 智能体
- 拿到回复后发回给你

这个管家可以运行在：
- 你家里的电脑
- 公司的服务器
- 树莓派（省电，适合 24 小时运行）
- 云服务器（VPS）

## 四、安装难吗？

### 实话实说

**如果你是技术小白**：需要一点耐心，跟着文档一步步来，大约 30 分钟能搞定。

**如果你有技术背景**：10-15 分钟，可能更快。

### 安装步骤（精简版）

#### 第 1 步：安装命令行工具

```bash
# Mac/Linux
curl -fsSL https://openclaw.ai/install.sh | bash

# Windows (PowerShell)
iwr -useb https://openclaw.ai/install.ps1 | iex
```

#### 第 2 步：运行新手引导

```bash
openclaw onboard
```

这一步会问你几个问题：
- 用哪个 AI 模型？（推荐用 OAuth 登录，免费额度够用）
- 要连接哪个聊天软件？（WhatsApp 扫码，Telegram/Discord 填 Token）
- 要不要开机自启动？（推荐要，这样 24 小时待命）

#### 第 3 步：连接你的聊天软件

**WhatsApp**：会显示一个二维码，用手机 WhatsApp 扫一下就行（和登录 WhatsApp Web 一样）

**Telegram**：需要创建一个 Bot（找 @BotFather 聊几句，1 分钟搞定）

**Discord**：创建一个应用，复制 Token 填进去

#### 第 4 步：测试

给你的 AI 发条消息："你好"

如果收到回复，恭喜，成功了！🎉

### 可能遇到的坑

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 收不到回复 | 消息被当成"陌生人"拦截了 | 运行 `openclaw pairing list` 查看并批准 |
| WhatsApp 掉线 | 网络不稳定 | 重新扫码登录 |
| AI 不响应 | 没配置 API Key | 运行 `openclaw onboard` 重新配置 |
| 命令找不到 | Node.js 版本太低 | 升级到 Node 22+ |

## 五、配置：让它听你的话

### 基础配置：谁能用它？

默认情况下，OpenClaw 对陌生消息比较警惕（这是好事）。你可以配置：

```json5
{
  // 只允许特定号码使用
  channels: {
    whatsapp: {
      allowFrom: ["+8613800138000", "+8613900139000"]
    }
  }
}
```

### 群聊配置：别让它乱说话

在群里，你肯定不想它每条消息都回复。可以设置：

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }  // 只有@它才回复
      }
    }
  },
  agents: {
    list: [{
      groupChat: { 
        mentionPatterns: ["@AI", "@助手", "帮帮忙"] 
      }
    }]
  }
}
```

这样，只有当有人 @AI 或者说"帮帮忙"时，它才会响应。

### 安全配置：别让它乱来

你可以限制它能做什么：

```json5
{
  agents: {
    defaults: {
      sandbox: {
        mode: "non-main",      // 非主会话用沙箱运行
        workspaceAccess: "ro"  // 只能读文件，不能写
      }
    }
  }
}
```

**沙箱**就像一个"隔离房间"，AI 在里面能做的事情有限制，就算出问题也不会影响你的电脑。

## 六、进阶玩法

### 1. 多智能体：一个管工作，一个管生活

你可以配置两个不同的 AI：

```json5
{
  agents: {
    list: [
      { id: "personal", default: true },  // 私人助手
      { id: "work" }                       // 工作助手
    ]
  },
  bindings: [
    { agentId: "personal", match: { channel: "whatsapp", accountId: "personal" } },
    { agentId: "work", match: { channel: "whatsapp", accountId: "work" } }
  ]
}
```

然后用两个不同的 WhatsApp 账号，一个连私人 AI，一个连工作 AI。

### 2. 定时任务：让它主动找你

```bash
# 每天早上 9 点提醒你开会
openclaw cron add --schedule "0 9 * * *" --message "提醒：9:30 有晨会"
```

### 3. 技能扩展：让它更强大

OpenClaw 有个技能市场（ClawHub），可以安装各种技能：

```bash
# 安装图片生成技能
clawhub install image-gen

# 安装网页搜索技能
clawhub install web-search
```

已发布的技能包括：
- 网页搜索
- 图片生成/编辑
- 文档总结
- 代码执行
- 语音合成
- 更多...

### 4. 手机远程控制

OpenClaw 有 iOS/Android 应用，配对后可以：
- 用手机摄像头拍照让 AI 分析
- 语音唤醒
- 查看 AI 的操作界面（Canvas）

## 八、隐私和安全

### 数据存在哪？

**都在你自己的设备上**。OpenClaw 只是桥梁，不存储你的聊天记录。

### 安全吗？

默认配置下：
- ✅ 陌生消息需要批准
- ✅ 群聊需要@才响应
- ✅ 敏感操作需要确认
- ✅ 支持沙箱隔离

### 需要注意什么？

1. **别把 Gateway 暴露在公网**（除非你懂网络安全）
2. **定期更新**（`openclaw update`）
3. **谨慎安装第三方技能**（可能有不安全代码）
4. **用强密码/Token**
